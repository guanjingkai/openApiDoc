# 小云积分
> `小云积分`是由`汉云优品`开发并运营的积分共享平台，让新生的数据垃圾[积分]得以更有效的利用，通过小云积分，可以将第三方平台积分导入到小云的`积分总线`系统，对积分进行了重新定义，并赋予了新的能量，可在与小云合作的实体商户进行商品/优惠券的兑换。
